## Usage

1. Navigate to the SQL to CSV Converter web app.
2. Input your SQL select result table in the given format:
```
+----+---------+--------+---------+
| ID |  Name   |  Age   |  Grade  |
+----+---------+--------+---------+
| 1  | Alice  |  20    |  B      |
| 2  | Bob    |  22    |  A      |
| 3  | Charlie|  21    |  C      |
| 4  | David  |  23    |  B      |
+----+---------+--------+---------+
```
3. Click the "Convert" button.
4. Receive your data in CSV format instantly.
